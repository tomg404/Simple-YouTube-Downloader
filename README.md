# Simple YouTube Downloader

Simple YouTube Downloader (sytd) is a small Python library for quickly downloading YouTube videos.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install `sytd`.

```bash
pip install sytd
```

## Usage
```bash
python -m sytd
```
